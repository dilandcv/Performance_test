# Performance_test
# how to run the program?
The program is run from menu.py and executes in the console

# functions:
The program has five functions: the first allows you to add students to an empty list; the second allows you to view the students on that list; the third provides information for a single student; the fourth function allows you to update student data; and the last allows you to delete students from the list.
